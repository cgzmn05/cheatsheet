HTML/CSS Cheatsheet

a little list to help future me with quick info about common HTML and CSS tags.

HTML Tags Description Attributes
<img> links images using URL; self-closing tag src="" is image source for URL to link image, alt="" is concise text description to help with accessability, width or height attributes can also be added
<video></video> to embed video content src="" for video URL, height, width, and similar to audio: controls, loop.
<audio></audio> to embed sound content src="" for the URL of the sound to embed, controls allows to have play/pause/etc buttons, loop to replay continuously.
<a></a> Stands for anchor and it's used to link a website. href="" is used to hide link and link it to text, target="_blank" opens link in new website.
